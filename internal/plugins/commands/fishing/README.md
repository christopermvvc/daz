# fishing command

Go fishing for a chance at cash. Results are sent via PM, with big wins announced publicly.

Config (optional)
- `cooldown_hours` (default: 2)

Notes
- Uses the economy API for bait debits and winnings credits.
- Default bait: ciggie (free).
- Baits: ciggie (free), worm ($2), servo pie ($5), lure ($15), prawn ($25), squid ($40).
